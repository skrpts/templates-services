## Application
ChatGPT Desktop (OpenAI)

## Features
- System-wide keyboard shortcut (Option+Space on macOS)
- Screenshot and screen sharing for visual context
- File upload and analysis
- GPT-4o, o1 model access (subscription dependent)

## Integration
- Limited programmatic integration compared to API
- No native MCP support currently
- Best used for ad-hoc tasks and visual analysis

## Notes
- Requires ChatGPT Plus or Team subscription for GPT-4
- macOS and Windows versions available
- Voice mode supported
